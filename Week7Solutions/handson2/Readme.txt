# Hands-On 2
## Objective:
Demonstrate usage of JSX, inline CSS, and DOM rendering in a React app.

## App Name:
officespacerentalapp

## Components:
- `OfficeList`: Displays office listings with conditional styling.
- `App`: Imports and renders `OfficeList`.

## Features:
- JSX usage for rendering elements.
- Looping through a list of office spaces.
- Conditional CSS for rent color.
- External CSS (`styles.css`) for styling.

## Output:
- Heading of the page.
- Image, name, rent, address for each office.
- Rent shown in red if ≤ ₹60,000, green if > ₹60,000.
