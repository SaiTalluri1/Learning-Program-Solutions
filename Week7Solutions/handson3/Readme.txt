# Week 7 - Hands-On 3: React Event Handling

## App Name: `eventexamplesapp`

### Components & Features

1. **Counter**
   - Increment & Decrement buttons
   - Displays "Hello" message

2. **SayWelcome**
   - Button that shows alert with custom message

3. **SyntheticEventExample**
   - Demonstrates synthetic events with `onClick`

4. **CurrencyConvertor**
   - Converts Rupees to Euro on form submit
