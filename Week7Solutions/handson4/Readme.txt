
# Event Examples App

This React application demonstrates various event handling techniques.

## Features

- **Increment/Decrement Counter**: Updates count and prints hello message.
- **Say Welcome Button**: Shows message using parameterized function.
- **Synthetic Event**: Button click logs "I was clicked".
- **Currency Converter**: Converts Indian Rupees to Euro.

## Screenshots

The app was tested with 5 event-related features, each captured as a screenshot with appropriate file structure.

## Technologies Used

- React
- JavaScript (ES6)
- JSX
