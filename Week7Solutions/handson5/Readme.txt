# BloggerApp Hands-On 5

This is a simple React app that demonstrates conditional rendering using three components:

- **Book Details**
- **Blog Details**
- **Course Details**

## Features

- Displays a list of books, blogs, and courses
- Uses different types of conditional rendering
- Shows how to use the `map()` function and `key` props
- Organized layout using basic CSS
